# KNN Analysis: Health Risk & Energy Consumption

## Project Overview
This project contains two different machine learning tasks using the **K-Nearest Neighbors (KNN)** algorithm:
- **Classification:** Predicting health risk levels.
- **Regression:** Estimating residential energy consumption.

## Key Features
- **Data Preprocessing:** Handled outliers and applied `StandardScaler` for normalization.
- **Hyperparameter Tuning:** Optimized the $K$ value using the **Elbow Method**.
- **Visualization:** Included error rate plots and decision insights.

## Technologies Used
- Python, Pandas, Scikit-learn
- Matplotlib, Seaborn

## Results
- **Classification Accuracy:** %92 (Example)
- **Regression $R^2$ Score:** 0.85 (Example)
